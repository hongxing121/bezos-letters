# 2008 [[贝佐斯]]致股东信

致股东们：

在这个动荡的全球经济中，我们的基本方法没有改变：埋头苦干，着眼长远，痴迷于客户。[[长期主义|长期思维]]放大了我们现有的能力，让我们能够做到原本无法想象的新事物。它为发明所必需的失败和迭代提供了空间，让我们得以自由地在未知领域开拓。追求即时满足——或那遥不可及的即时满足承诺——你多半会发现前面已经挤满了人。[[长期主义|长期导向]]与[[客户至上|客户痴迷]]之间有着极好的化学反应。如果我们能识别出一个客户需求，并且进一步确信这个需求是有意义且持久的，那么我们的方法就允许我们耐心地花上好几年来交付解决方案。[[逆向工作法|"从客户需求倒推"]]可以与"技能前推"方法做一个对比。所谓"技能前推"，就是利用现有技能和能力来驱动商业机会，其逻辑是："我们非常擅长X，那么X还能用来做什么？"这是一种有用且有回报的商业方法。然而，如果只用这一种方法，公司就永远不会被驱动去发展新技能。最终，现有的技能会过时。[[逆向工作法|从客户需求倒推]]往往要求我们习得新能力、锻炼新肌肉，尽管迈出第一步时会感到多么不适和笨拙。

[[Kindle]]是我们基本方法的一个好例子。四年多以前，我们从一个长期愿景出发：人类有史以来出版的每一本书，任何语言，都能在不到60秒内获取。我们设想的客户体验不允许在[[Kindle]]硬件设备和[[Kindle]]服务之间划出任何硬性界限——两者必须无缝融合。[[亚马逊]]此前从未设计或制造过硬件设备，但我们没有为了迁就当时现有的技能而修改愿景，而是招聘了一批才华横溢的（而且充满使命感的！）硬件工程师，开始学习一项新的组织技能——一项我们需要掌握的技能，以便在未来更好地服务读者。

我们感激且兴奋的是，[[Kindle]]的销售超出了我们最乐观的预期。2月23日，我们开始发货Kindle 2。如果你还没见过，Kindle 2保留了客户喜爱的原版Kindle的一切优点，只是更薄、更快，显示屏更清晰，电池续航更长，而且能容纳1500本书。你可以从超过25万种最受欢迎的书籍、杂志和报纸中选择。无线传输免费，不到60秒你的书就到手了。我们已经收到了数千封客户关于[[Kindle]]的反馈邮件——令人惊叹的是，其中26%包含"爱"这个字。

**客户体验的三大支柱**

在我们的零售业务中，我们坚定地相信客户看重低价、海量选择和快速便捷的配送，而且这些需求会长期保持不变。我们很难想象十年后的客户会想要更高的价格、更少的选择或更慢的配送。正是因为相信这些支柱的持久性，我们才有信心投资去强化它们。我们知道，现在投入的精力将在未来很长时间内持续产生回报。

我们的定价目标是赢得客户信任，而非最大化短期利润。我们坚信，这种定价方式是[[长期主义|长期来看]]增长总利润的最佳途径。我们单件商品可能赚得更少，但通过持续赢得信任，我们会卖出更多商品。因此，我们在整个产品线上提供低价。出于同样的原因，我们持续投资于免费配送项目，包括 [[Amazon Prime]]。客户是信息充分且精明的，他们在做购买决策时会评估总成本，包括运费。过去12个月，全球客户通过利用我们的免费配送服务节省了超过8亿美元。

我们不懈地专注于增加选择，既在现有品类中扩充，也在开拓新品类。自2007年以来，我们新增了28个品类。一个快速增长且不断给我惊喜的业务是我们的鞋类商店 Endless.com，于2007年上线。

快速可靠的配送对客户至关重要。2005年我们推出了 [[Amazon Prime]]。年费79美元，Prime会员可享受无限次免费两日达快递，升级至一日达只需3.99美元。2007年我们推出了[[FBA|Fulfillment by Amazon]]（亚马逊物流），这是一项面向第三方卖家的新服务。通过FBA，卖家将库存存放在我们的全球配送网络中，由我们代为拣货、打包并发送给终端客户。FBA商品可享受 [[Amazon Prime]] 和超级省运费——就像这些商品是亚马逊自有库存一样。这样一来，FBA既改善了客户体验，也带动了卖家销售。2008年第四季度，我们代表使用FBA的卖家发出了超过300万件商品——这是客户和卖家的双赢。

**审慎支出**

我们选择的以客户体验为核心的路径，要求我们具备高效的成本结构。对股东来说好消息是：我们看到了大量的改善空间。我们放眼所及（而且大家都在看），都能发现日本资深制造业专家所说的"muda"——即[[浪费]]。我觉得这太令人振奋了。我把它看作潜力——数年乃至数十年的可变和固定生产率提升，以及更高效、更高速、更灵活的资本支出。

我们的首要财务目标仍然是最大化长期[[自由现金流]]，并以高[[资本回报率|投资资本回报率]]实现这一目标。我们在 [[AWS|Amazon Web Services]]、第三方卖家工具、数字媒体、中国市场和新产品品类上大力投资。我们做出这些投资，是因为相信它们能够达到有意义的规模，并且跨越我们对回报率的高标准。

在世界各地，一群了不起的、富有创造力的、勤奋的亚马逊人正在把客户放在第一位。作为这个团队的一员，我深感自豪。感谢你们——我们的股东——的支持、鼓励，以及加入我们这段持续的冒险旅程。

一如既往，我附上1997年致股东信。即使变化的速度在加快，我们希望并相信，聚焦于那些不变的东西，将使我们受益匪浅。
