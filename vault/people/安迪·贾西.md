# 安迪·贾西

## 基本信息
- **全名**：Andrew R. Jassy（安德鲁·R·贾西）
- **角色**：AWS 创始团队领导者、AWS CEO，2021年7月接任亚马逊CEO
- **在信中被提及**：1 封（[[2020 贝佐斯致股东信]]）

## 在信件中的形象

安迪·贾西在24封股东信中仅在最后一封——[[2020 贝佐斯致股东信]]——中被明确提及，但他的影子贯穿了贝佐斯关于 [[AWS]] 的所有叙述。

### 隐形的建设者（2006-2019）

虽然贾西的名字直到2020年才出现，但 [[AWS]] 从 [[2006 贝佐斯致股东信]] 起就成为贝佐斯信件中的重要主题。贝佐斯在多封信中描述了 AWS 从"24万注册开发者"（2006年）到"一百万活跃客户"（2014年）再到"年化营收500亿美元"（2020年）的惊人增长轨迹。作为 AWS 的实际缔造者和领导者，贾西带领这项业务从一个"激进的想法"（[[2014 贝佐斯致股东信]]）成长为全球云计算的领导者。

贝佐斯多次称赞 AWS 的创新速度——从2010年的61项新功能到2015年的722项再到2017年的1,400项——这些成绩直接反映了贾西领导下 AWS 团队的执行力。

### 被点名的继承者（2020）

在 [[2020 贝佐斯致股东信]] 中，贝佐斯在信末正式介绍了贾西作为继任者，给予了极高的评价和深厚的信任。

## 贝佐斯评价原话

> "我要特别感谢[[安迪·贾西]]同意接任 CEO 一职。这是一个责任重大的艰难工作。Andy 才华横溢，有着最高的标准。我向你们保证，Andy 不会让宇宙把我们变得平庸。他会调动所需的能量，保持我们之所以特别的那些东西。这不会容易，但至关重要。我也预测这会令人满足，而且经常很有趣。谢谢你，Andy。"
> —— [[2020 贝佐斯致股东信]]

### 评价解读

贝佐斯对贾西的评价高度凝练，包含了几个关键要素：
1. **"才华横溢"**——对智识能力的认可
2. **"最高的标准"**——直接呼应 [[2017 贝佐斯致股东信]] 中对[[高标准]]的系统论述，这是贝佐斯眼中领导者最重要的品质之一
3. **"不会让宇宙把我们变得平庸"**——将贾西的使命与 [[2020 贝佐斯致股东信]] 的核心哲学主题"[[差异化即生存]]"直接关联，意味着贾西被视为能够持续对抗"熵增"、保持亚马逊独特性的人
4. **"令人满足，而且经常很有趣"**——暗示贾西不仅是严肃的执行者，也具有享受创业过程的心态

这段评价出现在贝佐斯整封信的最末尾，紧跟在"善良，保持原创，创造的比消耗的多"这句最终寄语之前，位置本身就表明了传承的庄重性。
