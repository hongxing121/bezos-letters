# AWS（Amazon Web Services）

## 基本信息
- **全称**：Amazon Web Services
- **类型**：自建业务
- **首次提及**：[[2006 贝佐斯致股东信]]
- **提及信件数**：12 封（2006-2020年间多次提及）

## 发展历程

- **2006年**：首次在信中提及。已提供十种不同的网络服务，建立超过24万注册开发者社区。围绕软件开发者这一全新客户群构建业务，提供存储和计算等基本需求。（[[2006 贝佐斯致股东信]]）
- **2008年**：被列为重点投资领域之一。（[[2008 贝佐斯致股东信]]）
- **2009年**：推出众多新服务，包括Amazon关系数据库服务、虚拟私有云、Elastic MapReduce等。在欧盟增加服务，新设北加州区域，计划进入亚太区域。吸引了有史以来最多的新客户，包括许多大型企业客户。（[[2009 贝佐斯致股东信]]）
- **2010年**：亚马逊工程师在数据管理方面的突破成为AWS云存储和数据管理服务的起点。S3、Elastic Block Store和SimpleDB源自亚马逊独有的技术。（[[2010 贝佐斯致股东信]]）
- **2011年**：发展到拥有30种不同的服务。S3存储超过9,000亿个数据对象，每天新增超过10亿个。S3日常处理超过每秒50万次事务。所有服务按需付费，自助式服务。（[[2011 贝佐斯致股东信]]）
- **2012年**：发布159项新功能和服务。自推出以来已27次降价。推出AWS Trusted Advisor帮助客户节省成本。（[[2012 贝佐斯致股东信]]）
- **2013年**：推出280项重要服务和功能（2010年61项、2011年82项、2012年159项）。全球10个区域、26个可用区、51个边缘节点。八年来降价超过40次。（[[2013 贝佐斯致股东信]]）
- **2014年**：超过一百万活跃客户。第四季度使用量同比增长约90%。推出516项新功能和服务。推出Amazon Aurora数据库。被贝佐斯列为三大"梦幻业务"之一。（[[2014 贝佐斯致股东信]]）
- **2015年**：年销售额即将突破100亿美元，达到这一里程碑的速度比亚马逊当年还快。提供超过70种服务。发布722项重要新功能和服务。已降价51次。12个地理区域33个可用区。（[[2015 贝佐斯致股东信]]）
- **2016年**：开放机器学习和AI服务，推出Amazon Lex、Polly、Rekognition等。（[[2016 贝佐斯致股东信]]）
- **2017年**：年化营收200亿美元。发布超过1,400项重要服务和功能。推出SageMaker。re:Invent大会现场参会者超过40,000人。（[[2017 贝佐斯致股东信]]）
- **2018年**：年化营收300亿美元。推出DynamoDB等专用数据库。SageMaker让AI民主化。（[[2018 贝佐斯致股东信]]）
- **2020年**：年化营收达到500亿美元。服务数百万客户。在新冠疫情中发挥关键作用，支持医疗、教育和远程工作。能效是传统企业数据中心的3.6倍。（[[2020 贝佐斯致股东信]]）

## 贝佐斯评价

> "Amazon Web Services是另一个例子。通过AWS，我们正在围绕一个全新的客户群体——软件开发者——构建一项新业务。" —— [[2006 贝佐斯致股东信]]

> "AWS本身——作为一个整体——就是一个例子。没有人要求过AWS。没有人。但事实证明，世界已经准备好了、渴望这样一种产品，只是不知道而已。" —— [[2018 贝佐斯致股东信]]

> "很多人在我们起步时说AWS是一个大胆——且不寻常的——赌注。'这和卖书有什么关系？'" —— [[2015 贝佐斯致股东信]]

> "所有AWS服务都是按需付费，从根本上将资本支出转化为可变成本。AWS是自助式的：你不需要谈判合同或联系销售人员。" —— [[2011 贝佐斯致股东信]]

## 体现的核心概念
- [[客户至上]] — 90%-95%的功能由客户需求驱动
- [[自助服务平台]] — 无守门人，任何人都能使用
- [[长期主义]] — 从零开始耐心培育至500亿美元业务
- [[发明与失败]] — 没有人要求过AWS，完全靠漫游和直觉
- [[漫游]] — 追随好奇心和直觉的非线性发现过程
